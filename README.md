Project Overview

This project predicts the Air Quality Index (AQI) using Machine Learning algorithms based on air pollutant data such as:

PM2.5
PM10
NO2
SO2
CO

The project includes:

Data preprocessing
Exploratory Data Analysis (EDA)
Data visualization using Seaborn & Matplotlib
Machine Learning model training
AQI prediction
Model evaluation

The main goal is to analyze pollution patterns and build a predictive model for estimating AQI values.

Features
Clean and preprocess AQI dataset
Handle missing values
Perform EDA and visualization
Correlation heatmap analysis
Train ML models
Predict AQI values
Evaluate model accuracy
Feature importance analysis

Linear Regression
